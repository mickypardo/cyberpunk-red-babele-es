# dungeon-world-babele-es

## Description / Descripción

This is Spanish translation of JasonAlanTerry Cyberpunk RED core module for Foundry VTT using Simone's Babeles module.

Stats, roles and skills description are based on the Cyberpunk RED Core Spanish edition por Holocubierta.

----

Esta es la traducción al Español del sistema Cyberpunk RED core para Foundry VTT, usando el modulo Babele de Simone.

Las descripciones de Características, roles y habilidades esta basado en la traducción realizada por Holocubierta y que se encuentra disponible en el material descargable de su página.
